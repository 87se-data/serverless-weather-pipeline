# 🌤️ Serverless Weather Data Pipeline

気象庁の生データ（GRIB2形式）を高速で解析し、Androidアプリへリアルタイムにストリーミング配信（SSE）する、完全サーバーレスなデータパイプラインです。極限までのコスト最適化と処理の高速化を両立しています。

## 🎯 Architecture Overview

システム全体の構成は以下の通りです。RDBや常時稼働サーバーを完全に排除し、**維持費ゼロ（ゼロスケール構成）**を実現しています。

1. **データ取得・解析 (Cython / Python)**
   * CythonによるC拡張を利用し、バイナリデータ（GRIB2）を高速パース。
2. **クラウドストレージ保存 (GCS)**
   * 解析済みデータを、クラウドネイティブな高圧縮フォーマット（Zarr / npy.zstd）に変換し、Google Cloud Storage (GCS) へ保存。
3. **API配信 (Cloud Run / FastAPI)**
   * GCSからデータをインメモリ展開し、FastAPIを用いてServer-Sent Events (SSE) で配信。
4. **クライアント (Android / Kotlin)**
   * スマホ実機上でリアルタイムにUIへ反映。

## 🚀 Key Achievements（実績）

* **劇的な処理速度の向上**
  Cythonの導入と並列処理の最適化。
* **FinOps（クラウドコスト最適化）**
  Cloud SQL（RDB）やGCE等の固定費がかかるインフラを廃止。GCSとCloud Runによる完全なステートレス設計へリファクタリングし、**インフラ維持費を実質ゼロに削減**。
* **AI-Driven Development**
  要件定義やアーキテクチャ設計は独自に行いつつ、実装レイヤーにおいてはLLM（Gemini等）を活用。開発スピードを圧倒的に加速させました。
